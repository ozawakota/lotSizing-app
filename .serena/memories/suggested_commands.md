# Suggested Commands

## Development
```bash
npm run dev          # Start dev server (with --host for network access)
npm run build        # TypeScript check + Vite build
npm run preview      # Preview production build
npm run lint         # ESLint check
```

## Deployment
```bash
npm run rebuild      # Build and copy dist to docs/ folder
npm run deploy       # Deploy to GitHub Pages via gh-pages
npm run git          # git add . && commit && push origin main
```

## System Utilities (Darwin/macOS)
```bash
ls, find, grep, cat  # Standard file utilities
git log, git diff    # Git history
open <file>          # Open file in default app
```
